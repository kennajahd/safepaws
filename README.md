# Safe Paws Rescue

Safe Paws Rescue is a fictional animal rescue organization website created to demonstrate HTML and CSS skills.

This site includes 6 pages with navigation, images, and an external style sheet.

## Features
- 6-page navigation system
- External CSS styling
- Google web font integration
- Images and layout design
- Email and external links
- Pseudo-class styling (:hover)

## Purpose
This project was created as part of a web design assignment to showcase knowledge of HTML and CSS.
